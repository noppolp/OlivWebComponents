Directive
- binding ng-model to file tag
usage: file-model="vm.image"
inspect: vm.image = {{ vm.image }}
- if true, act as loading based on loadingStyle

- hide : (default) replace "loading" to container's content

usage: loading-task="vm.loading"
inspect: vm.loading = {{ vm.loading }}
Content

- disable : disable all inputs in container

usage: loading-task="vm.loading" loading-style="disable"
inspect: vm.loading = {{ vm.loading }}
